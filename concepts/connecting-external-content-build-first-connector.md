# Quickstart: Create your first custom Microsoft Graph connector

<!-- 2. Introductory paragraph

Microsoft Graph connectors gives you the option to bring external data into Microsoft Graph and enhance Microsoft 365 intelligent experiences.

This .NET Core application shows how to use the Microsoft Graph connector API to create a custom connector to use with the Microsoft Search service and index custom items. The sample uses mock data, and indexes appliance parts inventory for Contoso Appliance Repair.
