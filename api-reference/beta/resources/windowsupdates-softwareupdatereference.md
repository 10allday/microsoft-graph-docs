# softwareUpdateReference resource type

Namespace: microsoft.graph.windowsUpdates

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

A reference to specific content. In a deployment, the same softwareUpdateReference could result in devices receiving different update revisions, but the content is considered contextually equivalent for all devices in the deployment.


Inherits from [deployableContent](../resources/windowsupdates-deployablecontent.md).

## Properties
None.

## Relationships
